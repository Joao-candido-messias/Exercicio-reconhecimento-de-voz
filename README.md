# Exercicio de Reconhecimento de Voz

Projeto web simples desenvolvido como exercício prático para estudo de reconhecimento de voz utilizando recursos nativos do navegador.

## Objetivo

Praticar o uso de APIs de voz em aplicações web, capturando a fala do usuário por meio do microfone e processando essas informações com JavaScript.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura do Projeto

Exercicio-reconhecimento-de-voz/
├── app/
├── index.html
├── style.css
└── README.md

## Como Executar

1. Clone o repositório:
git clone https://github.com/Joao-candido-messias/Exercicio-reconhecimento-de-voz.git

2. Acesse a pasta do projeto:
cd Exercicio-reconhecimento-de-voz

3. Abra o arquivo index.html em um navegador moderno.

## Funcionamento

- Solicita permissão de acesso ao microfone.
- Captura a voz do usuário.
- Processa o reconhecimento de voz via JavaScript.
- Exibe ou utiliza o resultado conforme a lógica implementada.

## Observações

- É necessário permitir o acesso ao microfone.
- Recomendado utilizar Google Chrome ou Microsoft Edge.
- Verifique o console do navegador em caso de erro.

## Aprendizados

- Manipulação do DOM
- Eventos em JavaScript
- Uso de APIs do navegador
- Captura de áudio
- Estrutura básica de aplicações web
